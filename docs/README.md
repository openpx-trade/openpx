# OpenPX docs

This is the OpenPX documentation site, built with [Mintlify](https://mintlify.com).

## Local preview

Install the Mintlify CLI once:

```bash
npm i -g mintlify
```

Then from this directory:

```bash
mintlify dev
```

## Structure

Three top-level tabs (`docs.json`):

- **API** — `api/`: getting started, per-exchange setup, unified methods, SDK guides.
- **WebSockets** — `websockets/`: connection, per-event update pages, session events.
- **Changelog** — `changelog.mdx`.

Other paths:

- `images/` — logos and favicon.
- `reference/types.mdx` — auto-generated by `just docs` from the JSON schema. Do not hand-edit.

## Deploys

Production deploys are handled by Mintlify's hosted git integration — connect this repo at [mintlify.com](https://mintlify.com) and pushes to `main` deploy automatically. There is intentionally no GitHub Actions deploy workflow in-repo.
