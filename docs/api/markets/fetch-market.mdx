---
title: "fetch_market"
description: "Fetch a single market by its native ID."
---

```rust
async fn fetch_market(market_id: &str) -> Result<Market, OpenPxError>
```

## Parameters

<ParamField path="market_id" type="string" required>
  | Exchange | Format |
  |----------|--------|
  | Kalshi | Ticker, e.g. `KXBTC-25MAR14-T20000` |
  | Polymarket | Condition ID, e.g. `0x311d0c4b…` |
</ParamField>

## Returns

[`Market`](/api/unified-schema#market) — the unified market struct.

## Example

<CodeGroup>

```rust Rust
let market = ex.fetch_market("KXBTC-25MAR14-T20000").await?;
```

```python Python
market = ex.fetch_market("KXBTC-25MAR14-T20000")
```

```typescript TypeScript
const market = await ex.fetchMarket("KXBTC-25MAR14-T20000");
```

</CodeGroup>
