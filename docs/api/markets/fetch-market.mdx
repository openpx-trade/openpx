---
title: "fetch_market"
sidebarTitle: "Fetch market"
openapi: GET /v1/markets/fetch_market
playground: simple
description: "Fetch a single market by its native ID."
---

## Parameters

<ParamField path="market_id" type="string" required>
  | Exchange | Format |
  |----------|--------|
  | Kalshi | Ticker, e.g. `KXBTC-25MAR14-T20000` |
  | Polymarket | Condition ID, e.g. `0x311d0c4b…` |
</ParamField>

## Returns

<ResponseField name="market" type="Market">
  <Expandable title="Market">
    <ResponseField name="id" type="string">Native ID — Kalshi ticker or Polymarket condition ID.</ResponseField>
    <ResponseField name="native_numeric_id" type="string?">Polymarket numeric DB id, for UI deep-links.</ResponseField>
    <ResponseField name="event_id" type="string?">Canonical OpenPX event ID grouping related markets.</ResponseField>
    <ResponseField name="title" type="string" />
    <ResponseField name="description" type="string" />
    <ResponseField name="slug" type="string?" />
    <ResponseField name="status" type="MarketStatus">`Active`, `Closed`, or `Resolved`.</ResponseField>
    <ResponseField name="outcomes" type="string[]" />
    <ResponseField name="token_id_yes" type="string?" />
    <ResponseField name="token_id_no" type="string?" />
    <ResponseField name="condition_id" type="string?" />
    <ResponseField name="volume" type="f64" />
    <ResponseField name="liquidity" type="f64?" />
    <ResponseField name="open_interest" type="f64?">Kalshi only.</ResponseField>
    <ResponseField name="open_time" type="DateTime?" />
    <ResponseField name="close_time" type="DateTime?" />
  </Expandable>
</ResponseField>

<RequestExample>

```rust Rust
let market = ex.fetch_market("KXBTC-25MAR14-T20000").await?;
```

```python Python
market = ex.fetch_market("KXBTC-25MAR14-T20000")
```

```typescript TypeScript
const market = await ex.fetchMarket("KXBTC-25MAR14-T20000");
```

</RequestExample>
