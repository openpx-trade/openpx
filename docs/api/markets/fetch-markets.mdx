```rust
async fn fetch_markets(
    params: &FetchMarketsParams,
) -> Result<(Vec<Market>, Option<String>), OpenPxError>
```

## Parameters

<ParamField path="limit" type="usize?">
  Per-page limit. Capped at 1000 (Kalshi) / 500 (Polymarket).
</ParamField>

<ParamField path="cursor" type="string?">
  Opaque cursor returned from a prior call.
</ParamField>

<ParamField path="status" type="MarketStatusFilter?">
  `active`, `closed`, `resolved`, or `all`. Defaults to `active`.
</ParamField>

<ParamField path="series_id" type="string?">
  Kalshi series ticker (e.g. `"KXBTC"`) or Polymarket series ID.
</ParamField>

<ParamField path="event_id" type="string?">
  Kalshi event ticker or Polymarket event ID/slug. When set, `series_id`,
  `cursor`, and `limit` are ignored.
</ParamField>

## Returns

<ResponseField name="markets" type="Market[]">
  One page of markets.

  <Expandable title="Market">
    <ResponseField name="id" type="string">Native ID — Kalshi ticker or Polymarket condition ID.</ResponseField>
    <ResponseField name="native_numeric_id" type="string?">Polymarket numeric DB id, for UI deep-links.</ResponseField>
    <ResponseField name="event_id" type="string?">Canonical OpenPX event ID grouping related markets.</ResponseField>
    <ResponseField name="title" type="string" />
    <ResponseField name="description" type="string" />
    <ResponseField name="slug" type="string?" />
    <ResponseField name="status" type="MarketStatus">`Active`, `Closed`, or `Resolved`.</ResponseField>
    <ResponseField name="outcomes" type="string[]">E.g. `["Yes", "No"]`.</ResponseField>
    <ResponseField name="token_id_yes" type="string?">Polymarket CLOB token for the Yes outcome.</ResponseField>
    <ResponseField name="token_id_no" type="string?">Polymarket CLOB token for the No outcome.</ResponseField>
    <ResponseField name="condition_id" type="string?">Polymarket condition ID.</ResponseField>
    <ResponseField name="volume" type="f64" />
    <ResponseField name="liquidity" type="f64?" />
    <ResponseField name="open_interest" type="f64?">Kalshi only.</ResponseField>
    <ResponseField name="open_time" type="DateTime?" />
    <ResponseField name="close_time" type="DateTime?" />
  </Expandable>
</ResponseField>

<ResponseField name="cursor" type="string?">
  Opaque pagination cursor. `null` on the last page.
</ResponseField>

## Example

<CodeGroup>

```rust Rust
use openpx::{FetchMarketsParams, MarketStatusFilter};

let params = FetchMarketsParams {
    status: Some(MarketStatusFilter::Active),
    limit: Some(100),
    ..Default::default()
};
let (markets, cursor) = ex.fetch_markets(&params).await?;
```

```python Python
result = ex.fetch_markets(status="active")
markets, cursor = result["markets"], result["cursor"]
```

```typescript TypeScript
const markets = await ex.fetchMarkets();
```

</CodeGroup>
