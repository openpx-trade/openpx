## Parameters

<ParamField path="limit" type="usize?">
  Per-page limit. Capped at 1000 (Kalshi) / 500 (Polymarket).
</ParamField>

<ParamField path="cursor" type="string?">
  Opaque cursor returned from a prior call.
</ParamField>

<ParamField path="status" type="MarketStatusFilter?">
  `active`, `closed`, `resolved`, or `all`. Defaults to `active`.
</ParamField>

<ParamField path="series_id" type="string?">
  Kalshi series ticker (e.g. `"KXBTC"`) or Polymarket series ID.
</ParamField>

<ParamField path="event_id" type="string?">
  Kalshi event ticker or Polymarket event ID/slug. When set, `series_id`,
  `cursor`, and `limit` are ignored.
</ParamField>

## Returns
