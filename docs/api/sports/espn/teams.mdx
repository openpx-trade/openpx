Returns ESPN's raw teams payload — schema varies by sport, so it's surfaced
as JSON rather than typed.

## Parameters

<ParamField path="league" type="string">League id (e.g. `"nfl"`).</ParamField>

## Returns

<ResponseField name="json" type="any">
  ESPN's `/{sport}/{league}/teams` payload. Typically includes a `sports[].leagues[].teams` array with team metadata, abbreviations, logos, and conference / division grouping.
</ResponseField>
