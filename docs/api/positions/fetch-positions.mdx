## Parameters

<ParamField path="market_id" type="string?">
  Filter to one market. Pass `None` for every open position.
</ParamField>

## Returns

<ResponseField name="positions" type="Position[]">
  <Expandable title="Position">
    <ResponseField name="market_id" type="string" />
    <ResponseField name="outcome" type="string" />
    <ResponseField name="size" type="f64" />
    <ResponseField name="average_price" type="f64" />
    <ResponseField name="current_price" type="f64" />
  </Expandable>
</ResponseField>

<RequestExample>

```rust Rust
let positions = ex.fetch_positions(None).await?;
```

```python Python
positions = ex.fetch_positions()
```

```typescript TypeScript
const positions = await ex.fetchPositions();
```

</RequestExample>
