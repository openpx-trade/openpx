For real-time books, use [WebSockets](/websockets/overview).

```rust
async fn fetch_orderbook(
    req: OrderbookRequest,
) -> Result<Orderbook, OpenPxError>
```

## Parameters

<ParamField path="market_id" type="string" required>
  Native market ID.
</ParamField>

<ParamField path="outcome" type="string?">
  `"Yes"` or `"No"` (binary markets).
</ParamField>

<ParamField path="token_id" type="string?">
  Polymarket CLOB token. Takes precedence over `outcome`.
</ParamField>

## Returns

<ResponseField name="orderbook" type="Orderbook">
  Bids sorted high-to-low, asks low-to-high.

  <Expandable title="Orderbook">
    <ResponseField name="market_id" type="string" />
    <ResponseField name="asset_id" type="string">Per-outcome (Kalshi: ticker, Polymarket: token).</ResponseField>
    <ResponseField name="bids" type="PriceLevel[]">Each: `{ price: f64, size: f64 }`.</ResponseField>
    <ResponseField name="asks" type="PriceLevel[]" />
    <ResponseField name="last_update_id" type="u64?">Sequence number when provided.</ResponseField>
    <ResponseField name="timestamp" type="DateTime?" />
    <ResponseField name="hash" type="string?">Polymarket book integrity hash.</ResponseField>
  </Expandable>
</ResponseField>

## Example

<CodeGroup>

```rust Rust
use openpx::OrderbookRequest;

let book = ex.fetch_orderbook(OrderbookRequest {
    market_id: "KXBTC-25MAR14-T20000".into(),
    outcome: Some("Yes".into()),
    token_id: None,
}).await?;
```

```python Python
book = ex.fetch_orderbook("KXBTC-25MAR14-T20000", outcome="Yes")
```

```typescript TypeScript
const book = await ex.fetchOrderbook("KXBTC-25MAR14-T20000", "Yes");
```

</CodeGroup>
