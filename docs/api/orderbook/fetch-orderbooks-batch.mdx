venues expose a batch book endpoint (Kalshi `/markets/orderbooks`,
Polymarket `/books`); this method exposes that as a unified primitive.

## Parameters

<ParamField path="market_ids" type="string[]" required />

## Returns

<ResponseField name="result" type="Orderbook[]">
  List of `Orderbook` results.

  <Expandable title="Orderbook">
    <ResponseField name="asks" type="PriceLevel[]" />
    <ResponseField name="asset_id" type="string" />
    <ResponseField name="bids" type="PriceLevel[]" />
    <ResponseField name="hash" type="string?">Exchange-provided hash for verifying book state integrity during replay. Polymarket: present on `book` snapshot events.</ResponseField>
    <ResponseField name="last_update_id" type="integer?" />
    <ResponseField name="market_id" type="string" />
    <ResponseField name="timestamp" type="DateTime?" />
  </Expandable>
</ResponseField>
