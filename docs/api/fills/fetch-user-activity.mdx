Polymarket only — Kalshi returns `NotSupported`. Returns unparsed JSON
from Polymarket's activity API.

## Parameters

<ParamField path="address" type="string" required>
  Wallet address to query.
</ParamField>

<ParamField path="limit" type="usize?">
  Max entries.
</ParamField>

## Returns

Raw JSON from Polymarket's activity API.

<RequestExample>

```rust Rust
use openpx::ExchangeInner;
use serde_json::json;
use openpx::FetchUserActivityParams;

let ex = ExchangeInner::new("polymarket", json!({
    "private_key": std::env::var("POLYMARKET_PRIVATE_KEY")?,
    "funder": std::env::var("POLYMARKET_FUNDER")?,
}))?;

let activity = ex.fetch_user_activity(FetchUserActivityParams {
    address: "0xabc…".into(),
    limit: Some(100),
}).await?;
```

```python Python
import os
from openpx import Exchange

ex = Exchange("polymarket", {
    "private_key": os.environ["POLYMARKET_PRIVATE_KEY"],
    "funder": os.environ["POLYMARKET_FUNDER"],
})

activity = ex.fetch_user_activity(address="0xabc…", limit=100)
```
