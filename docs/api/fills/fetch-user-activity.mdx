Polymarket only — Kalshi returns `NotSupported`. Returns unparsed JSON
from Polymarket's activity API.

## Parameters

<ParamField path="address" type="string" required>
  Wallet address to query.
</ParamField>

<ParamField path="limit" type="usize?">
  Max entries.
</ParamField>

## Returns

Raw JSON from Polymarket's activity API.

<RequestExample>

```rust Rust
use openpx::FetchUserActivityParams;

let activity = ex.fetch_user_activity(FetchUserActivityParams {
    address: "0xabc…".into(),
    limit: Some(100),
}).await?;
```

```python Python
activity = ex.fetch_user_activity(address="0xabc…", limit=100)
```

```typescript TypeScript
const activity = await ex.fetchUserActivity("0xabc…", 100);
```

</RequestExample>

<ResponseExample>

```json 200
{
  "address": "0xabc1234567890abcdef1234567890abcdef12345",
  "positions": [],
  "trades": [],
  "value": 0.0
}
```

</ResponseExample>
