## Parameters

<ParamField path="market_id" type="string" required>
  Native market ID.
</ParamField>

<ParamField path="outcome" type="string?">
  Outcome label. Kalshi infers Yes/No from the market.
</ParamField>

<ParamField path="token_id" type="string?">
  Polymarket CLOB token.
</ParamField>

<ParamField path="condition_id" type="string?">
  Polymarket condition ID for OI enrichment.
</ParamField>

<ParamField path="interval" type="PriceHistoryInterval" required>
  `1m`, `1h`, `6h`, `1d`, `1w`, or `max`.
</ParamField>

<ParamField path="start_ts" type="i64?">
  Unix seconds.
</ParamField>

<ParamField path="end_ts" type="i64?">
  Unix seconds.
</ParamField>

## Returns

`Candlestick[]`.

<ResponseField name="timestamp" type="DateTime<Utc>" />
<ResponseField name="open" type="f64" />
<ResponseField name="high" type="f64" />
<ResponseField name="low" type="f64" />
<ResponseField name="close" type="f64" />
<ResponseField name="volume" type="f64" />
<ResponseField name="open_interest" type="f64?">
  Kalshi only.
</ResponseField>

<RequestExample>

```rust Rust
use openpx::ExchangeInner;
use serde_json::json;
use openpx::{PriceHistoryRequest, PriceHistoryInterval};

let ex = ExchangeInner::new("kalshi", json!({
    "api_key_id": std::env::var("KALSHI_API_KEY_ID")?,
    "private_key_pem": std::env::var("KALSHI_PRIVATE_KEY_PEM")?,
}))?;

let candles = ex.fetch_price_history(PriceHistoryRequest {
    market_id: "KXBTC-25MAR14-T20000".into(),
    interval: PriceHistoryInterval::Hour,
    ..Default::default()
}).await?;
```
