## Parameters

<ParamField path="market_id" type="string" required>
  Native market ID.
</ParamField>

<ParamField path="market_ref" type="string?">
  Alternate ID (e.g. Polymarket `conditionId`) — preferred when present.
</ParamField>

<ParamField path="outcome" type="string?">
  Filter by outcome label.
</ParamField>

<ParamField path="token_id" type="string?">
  Filter by CLOB token (Polymarket).
</ParamField>

<ParamField path="start_ts" type="i64?">
  Inclusive lower bound (Unix seconds).
</ParamField>

<ParamField path="end_ts" type="i64?">
  Inclusive upper bound (Unix seconds).
</ParamField>

<ParamField path="limit" type="usize?">
  Max trades, exchange-capped.
</ParamField>

<ParamField path="cursor" type="string?">
  From a prior response.
</ParamField>

## Returns

<ResponseField name="trades" type="MarketTrade[]">
  <Expandable title="MarketTrade">
    <ResponseField name="trade_id" type="string?" />
    <ResponseField name="market_id" type="string" />
    <ResponseField name="asset_id" type="string">Per-outcome (Kalshi: ticker, Polymarket: token).</ResponseField>
    <ResponseField name="outcome" type="string?">`"Yes"` / `"No"`.</ResponseField>
    <ResponseField name="price" type="f64">Decimal in `[0, 1]`.</ResponseField>
    <ResponseField name="size" type="f64" />
    <ResponseField name="side" type="string?" />
    <ResponseField name="timestamp" type="DateTime<Utc>" />
  </Expandable>
</ResponseField>

<ResponseField name="cursor" type="string?">
  Pagination cursor; `null` on last page.
</ResponseField>

<RequestExample>

```rust Rust
use openpx::ExchangeInner;
use serde_json::json;
use openpx::TradesRequest;

let ex = ExchangeInner::new("kalshi", json!({
    "api_key_id": std::env::var("KALSHI_API_KEY_ID")?,
    "private_key_pem": std::env::var("KALSHI_PRIVATE_KEY_PEM")?,
}))?;

let (trades, cursor) = ex.fetch_trades(TradesRequest {
    market_id: "KXBTC-25MAR14-T20000".into(),
    limit: Some(100),
    ..Default::default()
}).await?;
```

```python Python
import os
from openpx import Exchange

ex = Exchange("kalshi", {
    "api_key_id": os.environ["KALSHI_API_KEY_ID"],
    "private_key_pem": os.environ["KALSHI_PRIVATE_KEY_PEM"],
})

result = ex.fetch_trades("KXBTC-25MAR14-T20000", limit=100)
trades, cursor = result["trades"], result["cursor"]
```

