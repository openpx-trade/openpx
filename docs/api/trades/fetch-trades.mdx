---
title: "fetch_trades"
description: "Public market trade tape."
---

```rust
async fn fetch_trades(
    req: TradesRequest,
) -> Result<(Vec<MarketTrade>, Option<String>), OpenPxError>
```

## Parameters

<ParamField path="market_id" type="string" required>
  Native market ID.
</ParamField>

<ParamField path="market_ref" type="string?">
  Alternate ID (e.g. Polymarket `conditionId`) — preferred when present.
</ParamField>

<ParamField path="outcome" type="string?">
  Filter by outcome label.
</ParamField>

<ParamField path="token_id" type="string?">
  Filter by CLOB token (Polymarket).
</ParamField>

<ParamField path="start_ts" type="i64?">
  Inclusive lower bound (Unix seconds).
</ParamField>

<ParamField path="end_ts" type="i64?">
  Inclusive upper bound (Unix seconds).
</ParamField>

<ParamField path="limit" type="usize?">
  Max trades, exchange-capped.
</ParamField>

<ParamField path="cursor" type="string?">
  From a prior response.
</ParamField>

## Returns

<ResponseField name="trades" type="MarketTrade[]">
  See [`MarketTrade`](/api/unified-schema#markettrade--fill--candlestick).
</ResponseField>

<ResponseField name="cursor" type="string?">
  Pagination cursor; `null` on last page.
</ResponseField>

## Example

<CodeGroup>

```rust Rust
use openpx::TradesRequest;

let (trades, cursor) = ex.fetch_trades(TradesRequest {
    market_id: "KXBTC-25MAR14-T20000".into(),
    limit: Some(100),
    ..Default::default()
}).await?;
```

```python Python
result = ex.fetch_trades("KXBTC-25MAR14-T20000", limit=100)
trades, cursor = result["trades"], result["cursor"]
```

```typescript TypeScript
const trades = await ex.fetchTrades("KXBTC-25MAR14-T20000");
```

</CodeGroup>
