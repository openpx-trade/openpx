---
title: "refresh_balance"
sidebarTitle: "Refresh balance"
openapi: POST /v1/balance/refresh_balance
playground: simple
description: "Force a refetch of cached balance / allowance state."
---

No-op on exchanges that don't cache. Polymarket uses this to refresh USDC
and CTF allowances after on-chain approvals.

## Parameters

None.

## Returns

`()` on success.

<RequestExample>

```rust Rust
use openpx::ExchangeInner;
use serde_json::json;

let ex = ExchangeInner::new("kalshi", json!({
    "api_key_id": std::env::var("KALSHI_API_KEY_ID")?,
    "private_key_pem": std::env::var("KALSHI_PRIVATE_KEY_PEM")?,
}))?;

ex.refresh_balance().await?;
```

```python Python
import os
from openpx import Exchange

ex = Exchange("kalshi", {
    "api_key_id": os.environ["KALSHI_API_KEY_ID"],
    "private_key_pem": os.environ["KALSHI_PRIVATE_KEY_PEM"],
})

ex.refresh_balance()
```

```typescript TypeScript
import { Exchange } from "openpx";

const ex = new Exchange("kalshi", {
  api_key_id: process.env.KALSHI_API_KEY_ID!,
  private_key_pem: process.env.KALSHI_PRIVATE_KEY_PEM!,
});

await ex.refreshBalance();
```
