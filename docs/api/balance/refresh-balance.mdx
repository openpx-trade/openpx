No-op on exchanges that don't cache. Polymarket uses this to refresh USDC
and CTF allowances after on-chain approvals.

```rust
async fn refresh_balance() -> Result<(), OpenPxError>
```

## Parameters

None.

## Returns

`()` on success.

## Example

<CodeGroup>

```rust Rust
ex.refresh_balance().await?;
```

```python Python
ex.refresh_balance()
```

```typescript TypeScript
await ex.refreshBalance();
```

</CodeGroup>
