## `fetch_trades`

```rust
async fn fetch_trades(
    req: TradesRequest,
) -> Result<(Vec<MarketTrade>, Option<String>), OpenPxError>
```

### Parameters

<ParamField path="market_id" type="string" required>
  Native market ID.
</ParamField>

<ParamField path="market_ref" type="string?">
  Alternate ID (e.g. Polymarket `conditionId`) — preferred when present.
</ParamField>

<ParamField path="outcome" type="string?">
  Filter by outcome label.
</ParamField>

<ParamField path="token_id" type="string?">
  Filter by CLOB token (Polymarket).
</ParamField>

<ParamField path="start_ts" type="i64?">
  Inclusive lower bound (Unix seconds).
</ParamField>

<ParamField path="end_ts" type="i64?">
  Inclusive upper bound (Unix seconds).
</ParamField>

<ParamField path="limit" type="usize?">
  Max trades, exchange-capped.
</ParamField>

<ParamField path="cursor" type="string?">
  From a prior response.
</ParamField>

`MarketTrade` carries `{ trade_id, market_id, asset_id, price, size, side?,
outcome?, fee_rate_bps?, exchange_ts? }`. See
[`MarketTrade`](/api/unified-schema#markettrade--fill--candlestick).
