## `fetch_fills`

```rust
async fn fetch_fills(
    market_id: Option<&str>,
    limit: Option<usize>,
) -> Result<Vec<Fill>, OpenPxError>
```

Returns your own executed fills, optionally filtered to one market.

`Fill`: `{ fill_id, order_id, market_id, outcome, side, price, size,
is_taker, fee, created_at }`.

## `fetch_user_activity`

Polymarket-only activity feed (positions, trades, portfolio data).

```rust
async fn fetch_user_activity(
    params: FetchUserActivityParams,
) -> Result<serde_json::Value, OpenPxError>
```

`FetchUserActivityParams { address: String, limit: Option<usize> }`. Returns
unparsed JSON from Polymarket's activity API.
