## `fetch_balance`

```rust
async fn fetch_balance() -> Result<HashMap<String, f64>, OpenPxError>
```

Returns a map keyed by currency / token symbol, e.g. `{ "USD": 1234.56 }` or
`{ "USDC": 120.00 }`.

## `fetch_balance_raw`

```rust
async fn fetch_balance_raw() -> Result<serde_json::Value, OpenPxError>
```

Returns the unparsed JSON. Useful when you need fields the unified surface
doesn't expose. Polymarket only — `NotSupported` on Kalshi.

## `refresh_balance`

```rust
async fn refresh_balance() -> Result<(), OpenPxError>
```

Forces a refetch of cached balance / allowance state. No-op on exchanges
that don't cache. Polymarket uses this to refresh USDC and CTF allowances
after on-chain approvals.
