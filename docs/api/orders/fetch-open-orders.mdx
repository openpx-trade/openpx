---
title: "fetch_open_orders"
description: "List open orders, optionally scoped to one market."
---

```rust
async fn fetch_open_orders(
    params: Option<FetchOrdersParams>,
) -> Result<Vec<Order>, OpenPxError>
```

## Parameters

<ParamField path="market_id" type="string?">
  Filter to one market. Pass `None` for every open order.
</ParamField>

## Returns

<ResponseField name="orders" type="Order[]">
  <Expandable title="Order">
    <ResponseField name="id" type="string" />
    <ResponseField name="market_id" type="string" />
    <ResponseField name="outcome" type="string" />
    <ResponseField name="side" type="OrderSide" />
    <ResponseField name="price" type="f64" />
    <ResponseField name="size" type="f64" />
    <ResponseField name="filled" type="f64" />
    <ResponseField name="status" type="OrderStatus">
      `pending`, `open`, `filled`, `partially_filled`, `cancelled`, or `rejected`.
    </ResponseField>
    <ResponseField name="created_at" type="DateTime<Utc>" />
    <ResponseField name="updated_at" type="DateTime<Utc>?" />
  </Expandable>
</ResponseField>
