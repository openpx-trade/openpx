## Parameters

<ParamField path="market_id" type="string" required>
  Native market ID.
</ParamField>

<ParamField path="outcome" type="string" required>
  `"Yes"` or `"No"`.
</ParamField>

<ParamField path="side" type="OrderSide" required>
  `buy` or `sell`.
</ParamField>

<ParamField path="price" type="f64" required>
  Decimal in `[0.0, 1.0]`.
</ParamField>

<ParamField path="size" type="f64" required>
  Contracts.
</ParamField>

<ParamField path="params.order_type" type="string">
  `gtc`, `ioc`, `fok`, or `market`. Defaults to `gtc`.
</ParamField>

<ParamField path="params.token_id" type="string">
  Polymarket only. Derived from `market_id` + `outcome` if omitted.
</ParamField>

## Returns

<ResponseField name="order" type="Order">
  <Expandable title="Order">
    <ResponseField name="id" type="string">Native order ID.</ResponseField>
    <ResponseField name="market_id" type="string" />
    <ResponseField name="outcome" type="string" />
    <ResponseField name="side" type="OrderSide">`buy` or `sell`.</ResponseField>
    <ResponseField name="price" type="f64">Decimal in `[0, 1]`.</ResponseField>
    <ResponseField name="size" type="f64" />
    <ResponseField name="filled" type="f64">Cumulative fills.</ResponseField>
    <ResponseField name="status" type="OrderStatus">
      `pending`, `open`, `filled`, `partially_filled`, `cancelled`, or `rejected`.
    </ResponseField>
    <ResponseField name="created_at" type="DateTime<Utc>" />
    <ResponseField name="updated_at" type="DateTime<Utc>?" />
  </Expandable>
</ResponseField>

```rust Rust
use openpx::OrderSide;
use std::collections::HashMap;

let mut params = HashMap::new();
params.insert("order_type".into(), "gtc".into());

let order = ex.create_order(
    "KXBTC-25MAR14-T20000",
    "Yes",
    OrderSide::Buy,
    0.55,
    10.0,
    params,
).await?;
```
