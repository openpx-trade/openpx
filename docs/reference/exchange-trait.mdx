---
title: "Exchange Trait"
description: "The unified API every supported exchange implements."
---

The `Exchange` trait, defined in `engine/core/src/exchange/traits.rs`, is the single interface every supported exchange implements. The SDK façade (`ExchangeInner`) dispatches to it via enum match — no `dyn` vtable.

## Method surface

| Method | Purpose |
|--------|---------|
| `id()` | Stable string identifier (`"kalshi"`, `"polymarket"`). |
| `name()` | Human-readable name. |
| `describe()` | `ExchangeInfo` capability flags (`has_websocket`, `has_create_order`, etc.). |
| `fetch_markets(params)` | Paginated list of markets. |
| `fetch_market(id)` | Single market by native ID. |
| `fetch_orderbook(req)` | L2 order book. |
| `fetch_price_history(req)` | OHLCV candles. |
| `fetch_trades(req)` | Recent market trades. |
| `fetch_orderbook_history(req)` | Historical book snapshots. |
| `create_order(...)` | Place an order. |
| `cancel_order(id, market)` | Cancel an order. |
| `fetch_order(id, market)` | Lookup one order. |
| `fetch_open_orders(params)` | List open orders. |
| `fetch_positions(market)` | Open positions. |
| `fetch_balance()` / `fetch_balance_raw()` | Wallet/account balance. |
| `fetch_fills(market, limit)` | Trade fills. |
| `fetch_user_activity(params)` | Activity feed (Polymarket). |
| `refresh_balance()` | Force a balance refetch (Polymarket). |

## Capability flags

Not every exchange supports every method. Always inspect `describe()` before calling — for example, `has_websocket` is `false` for unauthenticated Kalshi instances.
