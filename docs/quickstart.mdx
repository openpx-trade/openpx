## Fetch markets

<CodeGroup>

```rust Rust
use openpx::{ExchangeInner, FetchMarketsParams};
use serde_json::json;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let ex = ExchangeInner::new("kalshi", json!({}))?;
    let (markets, _cursor) = ex.fetch_markets(&FetchMarketsParams::default()).await?;
    println!("{} markets", markets.len());
    Ok(())
}
```

```python Python
import asyncio
from openpx import Exchange

async def main():
    ex = Exchange("kalshi", {})
    markets, _ = await ex.fetch_markets({})
    print(f"{len(markets)} markets")

asyncio.run(main())
```

```typescript TypeScript
import { Exchange } from "openpx";

const ex = new Exchange("kalshi", {});
const { markets } = await ex.fetchMarkets({});
console.log(`${markets.length} markets`);
```

</CodeGroup>

<Tip>
  Pass a config dict matching the exchange's expected fields — see [Kalshi](/exchanges/kalshi) or [Polymarket](/exchanges/polymarket) for the full schema.
</Tip>
