## Construct an `Exchange`

Pass an exchange ID and a config object. Both fields match the structure
documented on each exchange's setup page.

<CodeGroup>

```rust Rust
use openpx::ExchangeInner;
use serde_json::json;

#[tokio::main]
async fn main() -> anyhow::Result<()> {
    let ex = ExchangeInner::new("kalshi", json!({
        "api_key_id": std::env::var("KALSHI_API_KEY_ID")?,
        "private_key_pem": std::env::var("KALSHI_PRIVATE_KEY_PEM")?,
    }))?;
    println!("connected to {}", ex.name());
    Ok(())
}
```

```python Python
from openpx import Exchange
import os

ex = Exchange("kalshi", {
    "api_key_id": os.environ["KALSHI_API_KEY_ID"],
    "private_key_pem": os.environ["KALSHI_PRIVATE_KEY_PEM"],
})
print(f"connected to {ex.name}")
```

```typescript TypeScript
import { Exchange } from "openpx";

const ex = new Exchange("kalshi", {
  api_key_id: process.env.KALSHI_API_KEY_ID!,
  private_key_pem: process.env.KALSHI_PRIVATE_KEY_PEM!,
});
console.log(`connected to ${ex.name}`);
```

</CodeGroup>

## Fetch markets

`fetch_markets` returns one page plus an opaque cursor. Pass the cursor back
in to fetch the next page.

<CodeGroup>

```rust Rust
use openpx::FetchMarketsParams;

let (markets, next_cursor) = ex.fetch_markets(&FetchMarketsParams::default()).await?;
println!("{} markets, next cursor: {:?}", markets.len(), next_cursor);
```

```python Python
result = ex.fetch_markets()
print(f"{len(result['markets'])} markets, next cursor: {result['cursor']}")
```

```typescript TypeScript
const markets = await ex.fetchMarkets();
console.log(`${markets.length} markets`);
```

</CodeGroup>

<Note>
  The TypeScript binding's `fetchMarkets()` auto-paginates and returns every
  market in one call. Rust and Python expose the cursor directly.
</Note>
