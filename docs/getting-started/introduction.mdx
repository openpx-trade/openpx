OpenPX is an open-source, CCXT-style SDK for prediction markets. Trade
[Kalshi](/getting-started/exchanges/kalshi) and
[Polymarket](/getting-started/exchanges/polymarket) through a single `Exchange`
interface, in Rust, Python, or TypeScript.

## Why OpenPX

- **One interface, many venues.** The same calls return the same shapes across
  every supported exchange.
- **Speed first.** Written in Rust with `simd-json` decoding, tuned HTTP
  defaults, and zero-allocation hot paths.
- **Bring your own credentials.** OpenPX never custodies keys — you supply them
  at construction.

## Supported exchanges

| Exchange | REST | WebSocket | Notes |
|----------|------|-----------|-------|
