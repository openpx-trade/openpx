## Field crosswalk

| Unified field | Type | kalshi source | polymarket source | Notes |
|---|---|---|---|---|
| `market_ticker` | string | `MarketPosition.ticker` | `Position.slug` | **kalshi:** Kalshi market ticker; round-trips through `fetch_markets`. **polymarket:** Polymarket market slug; matches `Market.ticker`. |
| `outcome` | string | _synthetic_ | `Position.outcome` | **kalshi:** Title-cased from the sign of `position_fp` (positive → `Yes`, negative → `No`). **polymarket:** Outcome label as published by Polymarket. |
| `size` | number (double) | `MarketPosition.position_fp` | `Position.size` | **kalshi:** Absolute value of the signed `position_fp` FixedPointCount string. **polymarket:** Non-negative size in contracts. |
| `average_price` | number (double) | _synthetic_ | `Position.avgPrice` | **kalshi:** `total_traded_dollars / \|position_fp\|` — cost basis per contract. **polymarket:** Decimal probability in [0, 1]. |
| `current_price` | number (double) | _synthetic_ | `Position.curPrice` | **kalshi:** `market_exposure_dollars / \|position_fp\|` — mark per contract. **polymarket:** Decimal probability in [0, 1]. |
