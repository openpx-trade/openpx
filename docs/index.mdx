---
title: "OpenPX"
description: "A unified SDK for prediction markets."
---

OpenPX is an open-source, CCXT-style unified SDK for prediction markets, written in Rust with Python and TypeScript bindings. Bring your own exchange credentials and trade Kalshi and Polymarket through one trait.
