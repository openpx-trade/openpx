title: "Snapshot"
sidebarTitle: "🟣 Snapshot"
description: "Full L2 book — replace any cached state."
---

Emitted on initial subscribe and after any recovery. Replace any cached
book keyed by `(market_id, asset_id)`.

## Payload

<ResponseField name="market_id" type="string" />
<ResponseField name="asset_id" type="string" />
<ResponseField name="book" type="Orderbook">
  Full L2 book with sorted bids / asks.
</ResponseField>
<ResponseField name="exchange_ts" type="u64?">
  Exchange-authoritative millis when provided.
</ResponseField>
<ResponseField name="local_ts_ms" type="u64">
  Wall-clock millis at socket-read.
</ResponseField>
<ResponseField name="seq" type="u64" />

## Example

<CodeGroup>

```rust Rust
use px_core::WsUpdate;

while let Some(update) = updates.next().await {
    if let WsUpdate::Snapshot { market_id, book, .. } = update? {
        cache.insert(market_id, book);
    }
}
```

```python Python
from openpx import Snapshot

for update in ws.updates():
    if isinstance(update, Snapshot):
        cache[update.market_id] = update.book
```

```typescript TypeScript
await ws.onOrderbookUpdate("token-id", (err, update) => {
  if (update?.kind === "Snapshot") cache.set(update.market_id, update.book);
});
```

</CodeGroup>
