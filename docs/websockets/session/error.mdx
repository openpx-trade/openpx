---
title: "Error"
description: "Non-fatal connection error."
---

The session continues. Surface this for logging or alerting.

## Payload

<ResponseField name="message" type="string" />

## Example

<CodeGroup>

```rust Rust
use px_core::websocket::SessionEvent;

if let SessionEvent::Error { message } = event? {
    log::warn!("ws error: {message}");
}
```

```python Python
from openpx import SessionError

if isinstance(event, SessionError):
    print("ws error:", event.message)
```

```typescript TypeScript
ws.on("error", (message) => console.warn("ws error:", message));
```

</CodeGroup>
