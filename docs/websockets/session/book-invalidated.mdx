---
title: "BookInvalidated"
description: "A specific market's book is no longer trustworthy."
---

Discard your cache for `market_id`. The next [`Snapshot`](/websockets/snapshot)
will rebuild it.

## Payload

<ResponseField name="market_id" type="string" />
<ResponseField name="reason" type="InvalidationReason">
  `Reconnect`, `Lag`, `SequenceGap { expected, received }`, or `ExchangeReset`.
</ResponseField>

## Example

<CodeGroup>

```rust Rust
use px_core::websocket::SessionEvent;

if let SessionEvent::BookInvalidated { market_id, .. } = event? {
    cache.remove(&market_id);
}
```

```python Python
from openpx import BookInvalidated

if isinstance(event, BookInvalidated):
    cache.pop(event.market_id, None)
```

```typescript TypeScript
ws.on("bookInvalidated", ({ marketId }) => cache.delete(marketId));
```

</CodeGroup>
