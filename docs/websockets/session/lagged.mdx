---
title: "Lagged"
tag: "WSS"
description: "Slow consumer dropped messages."
---

The outbound dispatch channel overflowed. Followed by a
[`BookInvalidated`](/websockets/session/book-invalidated) with `reason: Lag`
for every affected market.

## Payload

<ResponseField name="dropped" type="u64" />
<ResponseField name="first_seq" type="u64" />
<ResponseField name="last_seq" type="u64" />

## Example

<CodeGroup>

```rust Rust
use px_core::websocket::SessionEvent;

if let SessionEvent::Lagged { dropped, .. } = event? {
    log::warn!("dropped {dropped} ws messages");
}
```

```python Python
from openpx import Lagged

if isinstance(event, Lagged):
    print("dropped", event.dropped)
```

```typescript TypeScript
ws.on("lagged", (dropped) => console.warn("dropped", dropped));
```

</CodeGroup>
