---
title: "Connected"
sidebarTitle: "🟣 Connected"
description: "Initial socket establishment."
---

Fires once per call to `connect()`. No payload.

## Example

<CodeGroup>

```rust Rust
use px_core::websocket::SessionEvent;

while let Some(event) = session.next().await {
    if matches!(event?, SessionEvent::Connected) {
        println!("connected");
    }
}
```

```python Python
from openpx import Connected

for event in ws.session_events():
    if isinstance(event, Connected):
        print("connected")
```

```typescript TypeScript
ws.on("connected", () => console.log("connected"));
```

</CodeGroup>
