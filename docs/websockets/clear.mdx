---
title: "Clear"
description: "Per-market book invalidation."
---

Drop everything ≤ this `seq` for the named market and wait for the next
`Snapshot`.

## Payload

<ResponseField name="market_id" type="string" />
<ResponseField name="asset_id" type="string" />
<ResponseField name="reason" type="InvalidationReason">
  `Reconnect`, `Lag`, `SequenceGap { expected, received }`, or `ExchangeReset`.
</ResponseField>
<ResponseField name="local_ts_ms" type="u64" />
<ResponseField name="seq" type="u64" />

## Example

<CodeGroup>

```rust Rust
use px_core::WsUpdate;

if let WsUpdate::Clear { market_id, .. } = update? {
    cache.remove(&market_id);
}
```

```python Python
if isinstance(update, Clear):
    cache.pop(update.market_id, None)
```

```typescript TypeScript
if (update?.kind === "Clear") cache.delete(update.market_id);
```

</CodeGroup>
