## Example

<CodeGroup>

```rust Rust
use px_core::WsUpdate;

if let WsUpdate::Fill { fill, .. } = update? {
    println!("filled {} @ {}", fill.size, fill.price);
}
```

```python Python
from openpx import Fill

    print(update.fill.size, update.fill.price)
```

}
