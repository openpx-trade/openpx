A public market trade. Not tied to one of your orders.

## Payload

<ResponseField name="market_id" type="string" />
<ResponseField name="asset_id" type="string" />
<ResponseField name="trade_id" type="string?" />
<ResponseField name="price" type="f64" />
<ResponseField name="size" type="f64" />
<ResponseField name="side" type="string?" />
<ResponseField name="aggressor_side" type="string?" />
<ResponseField name="outcome" type="string?">
  `"Yes"` / `"No"` (Polymarket).
</ResponseField>
<ResponseField name="fee_rate_bps" type="u32?" />
<ResponseField name="exchange_ts_ms" type="u64?" />

## Example

<CodeGroup>

```rust Rust
use px_core::WsUpdate;

if let WsUpdate::Trade { trade, .. } = update? {
    println!("{} @ {}", trade.size, trade.price);
}
```

```python Python
from openpx import Trade

if isinstance(update, Trade):
    print(update.trade.size, update.trade.price)
```

```typescript TypeScript
if (update?.kind === "Trade") {
  console.log(update.trade.size, update.trade.price);
}
```
