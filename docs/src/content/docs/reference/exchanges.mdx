## Supported Exchanges

### Kalshi

- **ID:** `kalshi`
- **Website:** [kalshi.com](https://kalshi.com)
- **API Docs:** [docs.kalshi.com](https://docs.kalshi.com)
- **Auth:** RSA key pair (`api_key_id` + `private_key_pem`)
- **Features:** Markets, Orders, Positions, Balance, Orderbook, Price History, Trades, Fills, WebSocket

### Polymarket

- **ID:** `polymarket`
- **Website:** [polymarket.com](https://polymarket.com)
- **API Docs:** [docs.polymarket.com](https://docs.polymarket.com/developers/)
- **Auth:** Private key + optional CLOB API credentials
- **Features:** Markets, Orders, Positions, Balance, Orderbook, Price History, Trades, Fills, User Activity, WebSocket

### Opinion

- **ID:** `opinion`
- **Website:** [opinion.trade](https://opinion.trade)
- **API Docs:** [docs.opinion.trade](https://docs.opinion.trade/developer-guide/opinion-open-api)
- **Auth:** API key + private key + multi-sig address
- **Features:** Markets, Orders, Positions, Balance, Orderbook, Price History, Trades, Fills, User Activity, WebSocket (requires auth)

## Configuration

All exchanges accept a JSON config object. Pass exchange-specific fields.

### Kalshi

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `api_key_id` | `string` | For auth | API key identifier |
| `private_key_pem` | `string` | For auth | RSA private key PEM content |
| `private_key_path` | `string` | For auth | Path to RSA private key PEM file (alternative to `private_key_pem`) |
| `demo` | `boolean` | No | Use demo environment (default: `false`). WebSocket is not available in demo mode |
| `api_url` | `string` | No | Custom API URL (default: `https://api.elections.kalshi.com/trade-api/v2`) |
| `verbose` | `boolean` | No | Enable verbose logging |

```json
{
  "api_key_id": "...",
  "private_key_pem": "...",
  "demo": false
}
```

### Polymarket

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `private_key` | `string` | For auth | Ethereum wallet private key (for order signing) |
| `funder` | `string` | No | Safe or Proxy wallet address (auto-detected from private key if omitted) |
| `api_key` | `string` | For trading | CLOB API key |
| `api_secret` | `string` | For trading | CLOB API secret |
| `api_passphrase` | `string` | For trading | CLOB API passphrase |
| `signature_type` | `string` | No | Signing mode: `"EOA"` (default), `"POLY_PROXY"`, or `"POLY_GNOSIS_SAFE"`. Auto-detected when possible |
| `chain_id` | `number` | No | Polygon chain ID (default: `137`) |
| `gamma_url` | `string` | No | Custom Gamma API URL for market metadata |
| `clob_url` | `string` | No | Custom CLOB API URL for trading |
| `data_api_url` | `string` | No | Custom Data API URL for historical data |
| `polygon_rpc_url` | `string` | No | Custom Polygon RPC endpoint |
| `builder_api_key` | `string` | No | Builder API key for affiliate attribution |
| `builder_secret` | `string` | No | Builder API secret |
| `builder_passphrase` | `string` | No | Builder API passphrase |
| `verbose` | `boolean` | No | Enable verbose logging |

```json
{
  "private_key": "0x...",
  "funder": "0x...",
  "api_key": "...",
  "api_secret": "...",
  "api_passphrase": "..."
}
```

### Opinion

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `api_key` | `string` | For auth | API key |
| `private_key` | `string` | No | Wallet private key |
| `multi_sig_addr` | `string` | No | Multi-sig wallet address |
| `api_url` | `string` | No | Custom API URL (default: `https://openapi.opinion.trade`) |
| `ws_url` | `string` | No | Custom WebSocket URL (default: `wss://ws.opinion.trade`) |
| `chain_id` | `number` | No | Blockchain chain ID (default: `56` for BSC) |
| `verbose` | `boolean` | No | Enable verbose logging |

```json
{
  "api_key": "...",
  "private_key": "0x...",
  "multi_sig_addr": "0x..."
}
```
