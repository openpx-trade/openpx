Real-time crypto price streaming via WebSocket for OpenPX.

## Overview

`px-crypto` provides real-time cryptocurrency price feeds via WebSocket connections, used for pricing and hedging prediction market positions.

## Installation

```toml
[dependencies]
px-crypto = "0.1"
```
