Real-time sports score streaming via WebSocket for OpenPX.

## Overview

`px-sports` provides real-time sports score feeds via the Polymarket Sports WebSocket, used for live sports-related prediction market data.

## Installation

```toml
[dependencies]
px-sports = "0.1"
```
